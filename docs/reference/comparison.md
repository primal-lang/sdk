# Comparison

## comp.eq

`comp.eq(a: Any, b: Any): Boolean`

Returns true if the two arguments are equal, false otherwise.

## comp.neq

`comp.neq(a: Any, b: Any): Boolean`

Returns true if the two arguments are not equal, false otherwise.

## comp.gt

`comp.gt(a: Any, b: Any): Boolean`

Returns true if the first argument is greater than the second, false otherwise.

## comp.lt

`comp.lt(a: Any, b: Any): Boolean`

Returns true if the first argument is less than the second, false otherwise.

## comp.ge

`comp.ge(a: Any, b: Any): Boolean`

Returns true if the first argument is greater than or equal to the second, false otherwise.

## comp.le

`comp.le(a: Any, b: Any): Boolean`

Returns true if the first argument is less than or equal to the second, false otherwise.
