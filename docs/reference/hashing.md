# Hashing

## Functions

### MD5

- **Signature:** `hash.md5(a: String): String`
- **Input:** One string
- **Output:** The MD5 hash of the string
- **Example:**

```
hash.md5("hello") // returns "5d41402abc4b2a76b9719d911017c592"
```

### SHA1

- **Signature:** `hash.sha1(a: String): String`
- **Input:** One string
- **Output:** The SHA1 hash of the string
- **Example:**

```
hash.sha1("hello") // returns "aaf4c61ddcc5e8a2dabede0f3b482cd9aea9434d"
```

### SHA256

- **Signature:** `hash.sha256(a: String): String`
- **Input:** One string
- **Output:** The SHA256 hash of the string
- **Example:**

```
hash.sha256("hello") // returns "2cf24dba5fb0a30e26e83b2ac5b9e29e..."
```

### SHA512

- **Signature:** `hash.sha512(a: String): String`
- **Input:** One string
- **Output:** The SHA512 hash of the string
- **Example:**

```
hash.sha512("hello") // returns "9b71d224bd62f3785d96d46ad3ea3d73..."
```
